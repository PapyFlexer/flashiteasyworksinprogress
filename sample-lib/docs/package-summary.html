Welcome to Flash'Iteasy API Documentation. This API enables developers to adapt their applications so they become Flash'Iteasy objects. While coding their functionalities, devs can also automatically generate the editors in the administration module. Moreover, custom editors can also be defined with this API, so specific contents can have complex editors. Please send any request to FIE Crew.
